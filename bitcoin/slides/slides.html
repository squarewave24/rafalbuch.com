What is Bitcoin

Bitcoin Transactions

  • Inputs  UTXO
  • Outputs
    • Require signatures 1-n
    • to spend > owner signs with private key
      • The contents of the transaction (including the hash of the previous transaction) are hashed and signed with senders private key.
      • senders public key is included in the transaction
      • there is a small program inside each transaction that gets executed to decide if a transaction is valid
        • Supports complex conditions such as Escrow, 2/3 signatures, Contracts
